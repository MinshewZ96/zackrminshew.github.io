About Myo Controlled Robotic Arm

Project Architecture

The purpose of this project is to show the cost-effective options of human prosthetics. We hope to achieve this by the implementation of an armband that measured EMG signals, a neural network, a 3d printed hand, and a LattePanda Delta. By using data collected from the armband and processed by MATLAB, we hope to mimic the motion of an individuals hand that is performing sign language. As of now, there is only the option to use up to 15 of the American Sign Language alphabeti letters shown below.
